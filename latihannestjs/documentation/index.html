Install nest@cli

Example :
$ npm i -g @nestjs/cli

Initial new Nest Project

Example :
$ nest new nama-project

Project setup

Example :
$ yarn install

Compile and run the project

Example :
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod

Run tests

Example :
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov

Generate Module

Example :
# nest cli global
$ nest g mo namaModule

# nest on project
$ yarn nest g mo namaModule

Generate Controller

Example :
# nest cli global
$ nest g co namaController

# nest on project
$ yarn nest g co namaController

Generate Service

Example :
# nest cli global
$ nest g s namaService

# nest on project
$ yarn nest g s namaService

Generate API Automatic

Example :
# nest cli global
$ nest g res namaapi

Prisma Database Migrate

Example :
# migrate for dev if you have shadow database
$ yarn prisma migrate dev

# migrate for production
$ yarn prisma migrate

Deployment

When you're ready to deploy your NestJS application to production, there are some key steps you can take to ensure it runs as efficiently as possible. Check out the deployment documentation for more information.

If you are looking for a cloud-based platform to deploy your NestJS application, check out Mau, our official platform for deploying NestJS applications on AWS. Mau makes deployment straightforward and fast, requiring just a few simple steps:

Example :
$ yarn install -g mau
$ mau deploy

With Mau, you can deploy your application in just a few clicks, allowing you to focus on building features rather than managing infrastructure.
